# Контактная Книга
Примитивное SPA Приложение

Данное приложение я реализировал полностью с Vue CLI и стилистикой преобразованной при помощи Bootstrap и FontAwesome. В качестве базы данных я использовал наиболее привычную мне Google Firebase. Все комментарии были размещены в файлах находящихся внутри папки "src".

Приложение основано на одном из личных практических проектов, представлявший из себя простой менеджер многопользовательских встреч, который я использовал для преобразования в еще более простую книгу контактов. Приложение содержит две страницы - отсортированный по фамилиям список имён, как и поле для добавления новых контактов, и отдел с поля для добавления описания и электронной почты.

База данных Google Firebase хранит параметры firstName и lastName, отвечающие за имя и фамилию контакта, а также description и email, отвечающие за описание и почту. Первые два параметра добавляются когда пользователь создаёт контакт. Поля ввода, которое запрашивает эти элементы, не будет работать, если один из них не заполнен. Нажати на кнопку с ведром удаляет контакт из базы данных. Вторые два параметра добавляются когда пользователь переходит по ссылке с информацией и вводит какую-либо информацию. Если параметры уже заданы - они автоматически заполняются в отдел ввода.


Не смотря на то, что мой опыт с JS-фреймворками в процессе усовершенствования, я преимущественно предпочитаю писать чистый код с минимальной или нулевой зависимостью от фреймворков.
